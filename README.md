# AGRINHO-2026

🌱 Agro Forte, Futuro Sustentável

Um projeto educativo interativo sobre agronegócio sustentável, desenvolvido com HTML, CSS e JavaScript puro.

📋 Descrição

O site apresenta informações importantes sobre práticas sustentáveis no agronegócio, com um design estilo jornal que explora o tema de forma indireta e envolvente. Inclui dois jogos educativos interativos para testar conhecimentos sobre sustentabilidade agrícola.

🎮 Jogos Inclusos

1. Jogo da Memória Agro

•
Encontre pares de práticas sustentáveis

•
8 cartas com 4 temas diferentes

•
Sistema de pontuação baseado em movimentos

•
Tema: 🌾 Rotação de Culturas, 💧 Gestão de Água, 🌍 Redução de Carbono, 🐝 Biodiversidade

2. Rotação de Culturas

•
Arraste as culturas para as estações corretas

•
Aprenda sobre o ciclo agrícola sustentável

•
Ganhe pontos ao acertar as combinações

•
Culturas: Milho (Primavera), Soja (Verão), Trigo (Outono), Aveia (Inverno)

🎨 Design

•
Paleta de Cores: Verde (sustentabilidade), Amarelo Pastel (esperança), Marrom (terra)

•
Responsivo: Funciona em desktop, tablet e mobile

•
Acessível: Código semântico e interface intuitiva
